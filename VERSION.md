### 打分公式说明
```
某环节实际得分 = Σ(score_j × weight_j)   [仅计权重>0的评委]
队伍决赛总分   = Σ(各环节实际得分)

其中：
  score_j   = 评委j对该队伍的打分（0~max_score）
  weight_j  = 评委j在该环节的权重（0~1小数）
  max_score = 该环节的满分分值

各环节满分之和 = 10+20+30+25+15 = 100，因此总分天然为百分制（0~100）
不做任何额外归一化，加权和直接就是最终得分。

示例验证（workshop，满分10分）：
  评委A：打分8，权重0.3 → 8×0.3=2.4
  评委B：打分9，权重0.5 → 9×0.5=4.5
  评委C：未设置权重 → 不参与
  该环节得分 = 2.4 + 4.5 = 6.9
```
